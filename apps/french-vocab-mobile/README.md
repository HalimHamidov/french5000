# French 5000 — Mobile Vocab App

A production-ready, local-first vocabulary app for drilling 5,000 frequent French words with spaced repetition.

Separate from the Telegram bot (`apps/french-vocab-bot/`). Both apps share the same source JSON data from `data/processed/`.

---

## Architecture

```
apps/french-vocab-mobile/
├── www/                         ← Web root (served as-is)
│   ├── index.html               ← App shell (single page)
│   ├── css/style.css            ← All styles, mobile-first
│   ├── js/
│   │   ├── app.js               ← Main entry: bootstrap, event wiring, rendering
│   │   ├── loader.js            ← JSON loading + schema auto-mapping
│   │   ├── scheduler.js         ← SRS intervals, session building, stats
│   │   └── storage.js           ← localStorage persistence
│   └── data/                    ← JSON copies (generated by prepare.js)
├── scripts/
│   └── prepare.js               ← Copies JSON from data/processed/ → www/data/
├── android/                     ← Capacitor Android project (after cap add android)
├── capacitor.config.json
├── package.json
└── README.md
```

**Data flow:**  `data/processed/*.json` → `scripts/prepare.js` → `www/data/` → `fetch()` in browser

**Persistence:**  All progress is saved in `localStorage` under `fr5k_*` keys — no server required.

## Quick Start (browser / dev)

```bash
cd apps/french-vocab-mobile
npm install
npm run dev
# Opens http://localhost:3000
```

`npm run dev` runs `prepare.js` (copies JSON data) then starts a static file server.

## Build Android APK

### Prerequisites

- Node.js 18+
- Java 17+ (JDK)
- Android Studio + Android SDK (API 33+)
- `ANDROID_HOME` environment variable set

### Steps

```bash
# 1. Install dependencies
cd apps/french-vocab-mobile
npm install

# 2. Copy data files into www/data/
node scripts/prepare.js

# 3. Add Capacitor Android (first time only)
npm run cap:add-android

# 4. Sync web assets into the Android project
npm run cap:sync

# 5. Open in Android Studio and build
npm run cap:open
# OR build APK directly from CLI:
cd android && ./gradlew assembleDebug
```

Debug APK output path:
```
android/app/build/outputs/apk/debug/app-debug.apk
```

Install on connected device:
```bash
adb install android/app/build/outputs/apk/debug/app-debug.apk
```

## JSON Data Mapping

Source: `data/processed/french_frequency_dictionary.json` (5,000 words)

| Source field       | Used as              |
|--------------------|----------------------|
| `id`               | Word ID (primary key)|
| `rank`             | Frequency rank 1–5000|
| `word`             | Display form         |
| `lemma`            | Base form (fallback) |
| `ipa`              | IPA transcription    |
| `pos`              | Part of speech       |
| `translation_ru`   | Russian translation  |
| `translation_en`   | English translation  |
| `example_fr`       | French example       |
| `example_ru`       | Russian example      |

Source: `data/processed/french_thematic_vocabulary.json` (27 themes)

| Structure          | Used as              |
|--------------------|----------------------|
| Top-level keys     | Theme names          |
| `.title`           | Theme display name   |
| `.words[]`         | Words in the theme   |

## Differentiation from Telegram Bot

| Aspect              | Telegram Bot (`apps/french-vocab-bot/`) | Mobile App (`apps/french-vocab-mobile/`) |
|---------------------|-----------------------------------------|------------------------------------------|
| Interface           | Chat messages + inline buttons          | Native-like card UI                      |
| Platform            | Any Telegram client                     | Android (+ any browser)                 |
| Data store          | SQLite via aiosqlite                    | localStorage                            |
| Reminders           | APScheduler broadcast                  | Web Notifications API                   |
| Language            | Python 3 + python-telegram-bot          | Vanilla JS (ES modules)                 |
| Data source         | `data/processed/*.json` (shared)        | `data/processed/*.json` (shared)        |
